How can we use the ALTER command in table?

~Alter command is used for changing the structure of the table.
~We can ADD,MODIFY and DROP the column using alter command.

Syntax:-

To add column:- ALTER table table_name add(col1 datatype,col2 datatype,..... ..,);
To drop column:- ALTER table table drop column column_name;
To modify a column:- ALTER table table modify column column_name;
