How can we use PRIMARY KEY constraint in table level contraints (TLC) and how can we use it with AlTER command?

~Whenever the primary key contains multiple columns it has to be specified at Table level.
~SQL PRIMARY KEY with ALTER TABLE : Most of the time, Primary Key is defined during the creation of the table but sometimes the Primary key may not be created in the already existing table. We can however add Primary Key using Alter Statement.

Syntax:-

Table level constraint:-create table table_name (col1 datatype ,col2 datatype,........,primary key(col1,col2..));
To add PRIMARY KEY:- ALTER table table add primary key(col_name);
To drop PRIMARY KEY :- ALTER table table drop primary key;
