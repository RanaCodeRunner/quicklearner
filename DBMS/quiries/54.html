Transaction Control Language(TCL) commands(COMMIT and ROLLBACK)

      Transaction Control Language (TCL) is used to manage transactions in a database. TCL statements are mainly used to manage the changes made by DML statements. It also allows statements to be grouped together into logical transactions.

COMMIT and ROLLBACK

            The changes made by the user are not physically written to the database table, giving only the user a view of his/her work, while the other users having access to these tables, continue to get the old information.

The COMMIT statement is used to make changes to data (inserts, updates, deletes) permanent to the database.

            The user can continue with any number of inserts, updates and/or deletion, and still can undo the work by issuing the ROLLBACK statement. The ROLLBACK statement is used to discard parts or all the work the user has done in the current transaction ROLLBACK is important in case an error is detected during the DML operations.

            SQL *Plus has the facility to automatically commit all the work, without explicitly issuing the COMMIT command. This is done by setting AUTOCOMMIT feature ON.

SET AUTOCOMMIT ON:Enable autocommit features

SET AUTOCOMMIT OFF:Is the default and disable automatic commiting

1. IMPLICIT COMMIT

The actions that will force a commit to occur, even without issuing the COMMIT command or without setting AUTOCOMMIT features ON are:

(i) Exiting SQL *Plus
(ii) Creating a database object like: table, view, index etc.
(iii) Granting or Revoking resources
(iv) Altering objects
(v) Connecting or disconnecting from Database server
Using any of these commands implicitly performs the COMMIT operation.

Syntax:-

COMMIT;

2. AUTO ROLLBACK

After completion of any DML statement, if the user experiences serious difficulty such as hardware failure and no explicit commit is made, SQL will automatically ROLLBACK all un-committed work. If the machine or database goes down, it does this as cleanup work the next time the database is brought up.

Syntax:-

ROLLBACK;
